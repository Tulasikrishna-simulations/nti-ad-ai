How it works

Geometry + product knowledge = inconsistency detection

The system doesn't just measure — it compares the measured geometry against the actual documented dimensions of the specific object in the frame.

Two-stage verification

H_est = (P_obj / P_ref) × H_ref Inconsistency Ratio = H_est / H_expected
