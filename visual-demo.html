How it works

The visual geometry formula

H_est — Estimated Object Height

H_est = (P_obj / P_ref) × H_ref
P_objPixel length of the subject being measured P_refPixel length of the known reference object H_refReal-world height of the reference object (metres) H_estEstimated real-world height of the subject

The system then compares H_est against expected real-world ranges. For adult humans, normal height is 1.5–2.0m. Significant deviation — an influencer appearing to be 2.4m against a standard door, or a product appearing much larger than its real dimensions — flags a potential geometric inconsistency in the advertisement.
